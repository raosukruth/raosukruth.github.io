# raosukruth.github.io

#GitHub Pages Portfolio

A simple portfolio landing page for `raosukruth.github.io` with editable featured project cards.

## Files

- `index.html`: page structure
- `styles.css`: visual theme and responsive layout
- `script.js`: featured project data and card rendering

## Customize Projects

Edit the `projects` array in `script.js`:

- `title`: card title
- `description`: short summary
- `href`: repository/demo URL
- `tags`: list of tech or topic labels

## Deploy to GitHub Pages

1. Create a repository named `raosukruth.github.io` (public).
2. Copy these files into the repo root.
3. Commit and push to the `main` branch.
4. Open `https://raosukruth.github.io` after deployment finishes.

For a project page instead of a user page, keep this same code in any repo and enable GitHub Pages from that repo's **Settings -> Pages**.
